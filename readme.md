Bibnet
=======================
Fun way for children to discover books

## Live version
[app.bieblo.be](http://app.bieblo.be/ "Bieblo live version")

## Dependencies
- npm
- gulp => `npm install -g gulp`
- bower => `npm install -g bower`

## Usage
- `npm install`
- `bower install`
- `gulp`
- Server running on `http://localhost:8080`
